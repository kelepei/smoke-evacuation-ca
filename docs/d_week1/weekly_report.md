## 4. 本周遇到的最大问题

当前最大问题是可视化和分析模块所需的数据接口尚未最终统一。

需要后续与其他成员确认：

1. 地图坐标采用 `(row, column)` 还是 `(x, y)`；
2. 坐标原点和坐标方向；
3. 行人状态和排队状态的统一枚举；
4. 目标出口和实际出口的输出方式；
5. 烟雾浓度和能见度的单位；
6. 全场烟雾数据的记录频率；
7. 社会关系边和关系权重的表达方式；
8. 从众行为和错误信息的事件记录方式；
9. 局部密度由模型模块还是分析模块计算；
10. 批量实验的统一配置格式。

如果这些接口在模型开发后再统一，可能导致可视化和日志模块重复修改，因此建议在第 3—4 周开始前进行一次接口确认。
