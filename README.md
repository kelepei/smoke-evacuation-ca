# smoke-evacuation-ca
烟雾风险感知与社会信息传播耦合的行人疏散元胞自动机模型及仿真平台
一.项目简介
本项目旨在构建一个基于元胞自动机的行人疏散仿真平台，综合考虑烟雾扩散、风险感知、社会关系和信息传播等因素，对火灾等紧急情况下的人群疏散行为进行模拟分析，
并采用模块化设计，实现地图导入、烟雾传播、行人决策、社会信息传播、疏散控制以及可视化展示等功能，为疏散策略研究和安全管理提供仿真支持。
二.项目目标
1. 构建一个烟雾风险感知与社会信息传播耦合的行人疏散元胞自动机模型；
2. 开发一套可视化疏散仿真平台，实现地图导入、人数设置、烟雾扩散、人员关系设置、信息传播和管控措施配置等功能；
3. 实现不同烟雾浓度、不同人群规模、不同社会关系和不同管控策略下的疏散仿真；
4. 形成疏散时间、出口利用率、拥堵区域、风险暴露人数、滞留人数等评价指标；
