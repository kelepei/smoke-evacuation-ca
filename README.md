# smoke-evacuation-ca
烟雾风险感知与社会信息传播耦合的行人疏散元胞自动机模型及仿真平台

一.项目简介

本项目旨在构建一个基于元胞自动机的行人疏散仿真平台。

综合考虑烟雾扩散、风险感知、社会关系和信息传播等因素，对火灾等紧急情况下的人群疏散行为进行模拟分析。

采用模块化设计，实现地图导入、烟雾传播、行人决策、社会信息传播、疏散控制以及可视化展示等功能，为疏散策略研究和安全管理提供仿真支持。

二.项目目标

- 构建二维网格化建筑环境模型
- 实现 CSV / JSON 地图导入
- 建立元胞自动机疏散模型
- 建立烟雾扩散模型
- 建立风险感知模型
- 建立社会关系网络
- 建立社会信息传播模型
- 实现疏散引导策略
- 开发仿真可视化平台
- 输出实验分析结果
